- 🛰️ &nbsp;**Systems Engineer @ WhiteBeard AI** — trading orchestration, market-making, low-latency infra
- 🫀 &nbsp;**Cofounder @ Vena** — a coin-sized health sensor that clips into analog watches
- 🧰 &nbsp;I write whatever the system actually needs — from a C++ market maker to a Next.js 3D site
